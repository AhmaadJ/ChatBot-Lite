About ChatBot Lite

ChatBot Lite is a website chatbot that allows you to receive answers to common questions, similar to a virtual assistant.

ChatBot Functions Include:

  • Getting today's date
  • Getting the current time
  • Launch a website/URL

New Additions:

  • Improved and more aesthetically pleasing site!
  • Link to ChatBot Lite V1 website

Future Additions:

  • Set reminders using Google Calendar
  • Weather Support
  • Statistical Functions

Window:

Click on the main.py file & click on the "Run" icon in order to get started!

Current Challenges

  • Currently, Chatbot Lite uses on online interpreter to embed the Python code into HTML. This causes some of the Chatbot functions to rely on the location of the server that the online interpreter is in rather than the location of the computer used to launch the site.

Try it on your own!

  • Copy the chatbotLite.py code into a new Python file in order to try the application on your own computer! This also solves the online interpreter location problem as the function will use your own computer's location.
