ChatBot Lite V3

About ChatBot Lite

ChatBot Lite is a website chatbot that allows you to receive answers to common questions, similar to a virtual assistant.

ChatBot Functions Include:

  • Getting today's date
  • Getting the current time
  • Launch a website/URL

New Additions:

  • Added Navigation Bar using Bootstrap
  • Uses JavaScript (Node.js) for chatbot code instead of Python now!
  • Link to ChatBot Lite V2 website
  • New Background Color

Future Additions:

  • Set reminders using Google Calendar
  • Weather Support
  • Statistical Functions

Window:

Click on the "Run" icon in order to get started!

Current Challenges

  • Currently, Chatbot Lite uses on online interpreter to embed the JavaScript code into HTML. This causes some of the Chatbot functions to rely on the location of the server that the online interpreter is in rather than the location of the computer used to launch the site. (Example: The time and URL)

Try it on your own!

  • Copy the chatbotLite.js code into a new JavaScript file in order to try the application on your own computer! This also solves the online interpreter location problem as the function will use your own computer's location.
